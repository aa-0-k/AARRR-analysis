# AARRR-analysis

# 📊 AARRR 모델 기반 고객 행동 분석 프로젝트  
이 프로젝트는 Olist 데이터셋을 활용하여 AARRR 모델 기반으로 고객 행동을 분석하는 프로젝트입니다.

## 📌 분석 목표  
✔ 신규 고객 획득 분석 (Acquisition)  
✔ 활성화 단계 분석 (Activation)  
✔ 유지율 분석 (Retention)  
✔ 추천 데이터 분석 (Referral)  
✔ 수익 모델 분석 (Revenue)  

## 🛠 사용 기술  
- **Python** (Pandas, Matplotlib, Seaborn)  
- **SQL** (MySQL)  
- **Jupyter Notebook**
